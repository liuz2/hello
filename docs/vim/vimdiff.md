# vimdiff

vimdiff 的基本用法：

```sh
$ vimdiff FILE_LEFT FILE_RIGHT
$ vimdiff -d FILE_LEFT FILE_RIGHT // 左右竖屏
$ vimdiff -o FILE_LEFT FILE_RIGHT // 上下横屏
```

可以使用快捷键在各个差异点之间快速移动：

- `]c` 跳转到下一个差异点
- `[c` 跳转到上一个差异点
