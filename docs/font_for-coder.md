# 适合编程的字体

1. [JetBrains Mono][1], A free and open source typeface for developers

## 如何在 VSCode 中设置字体？

以 `JetBranins Mono` 为例，修改 `settings.json`：

```json
{
    "editor.fontFamily": "JetBrains Mono",
    "editor.fontSize": 13,
    "editor.fontLigatures": true
}
```
