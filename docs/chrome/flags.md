# 有用的 flags

进入 flags 设置页面：`chrome://flags`

开启标签页分组：`chrome://flags/#tab-groups-collapse`

标签页预览：`chrome://flags/#tab-hover-card-images`

二维码快速分享：`chrome://flags/#sharing-qr-code-generator`
